## Functionality Analysis
1. 點按任一數字，數字會出現在output面板上。
2. 後點按四則運算符號會把之前點按的數字推上Output面板上部，連帶顯現數字以及四則運算符號。
3. 接著再按「=」符號完成四則運算操作，在output面板上顯示結果。
4. 按下「AC」可以清除面板上的全部數字。
5. 按下「DEL」可以清除一個數字。
6. 「.」在一個數字裡最多只會出現一次。
